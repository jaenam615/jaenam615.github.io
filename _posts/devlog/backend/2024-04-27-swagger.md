## Swagger

일반적으로 개발을 할 떄 프론트엔드 개발자와 백엔드 개발자가 협업하는 방식으로 진행되는데,  
클라이언트와 서버 사이에서 어떤 방식으로 데이터를 상호 전달할지에 대한 명세가 필요하다.  
스웨거(Swagger)는 이럴 때 API 기능을 명세하고 바로 테스트 할 수 있도록 도와주는 대표적인 API 관리 도구이다.
